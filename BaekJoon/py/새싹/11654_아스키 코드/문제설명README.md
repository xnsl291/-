### 11654 아스키 코드 
***

##### 문제 설명

알파벳 소문자, 대문자, 숫자 0-9중 하나가 주어졌을 때, 주어진 글자의 아스키 코드값을 출력하는 프로그램을 작성하시오.
##### 입력
알파벳 소문자, 대문자, 숫자 0-9 중 하나가 첫째 줄에 주어진다.

##### 출력
입력으로 주어진 글자의 아스키 코드 값을 출력한다.

##### 예제입력1
```
A
```
##### 예제출력1
```
65
```

##### 예제입력2
```
C
```
##### 예제출력2
```
67
```

##### 예제입력3
```
0
```
##### 예제출력3
```
48
```

##### 예제입력4
```
9
```
##### 예제출력4
```
57
```

##### 예제입력5
```
a
```
##### 예제출력5
```
97
```

##### 예제입력6
```
z
```
##### 예제출력6
```
122
```
